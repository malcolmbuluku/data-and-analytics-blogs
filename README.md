## Usage
1. Open `index.html` in a web browser to view the homepage
2. Navigate to articles using the post cards
3. Toggle dark mode using the theme button
4. All social media links are placeholders - update with actual URLs

## Customization
To customize the blog:
1. Update social media links in footer sections
2. Add more articles following the existing templates
3. Modify color scheme in CSS variables (`:root` section)
4. Update newsletter form action URL
5. Add actual contact information
